# Operation_System

同济大学《操作系统》课程资料仓库——课程设计课件、作业与参考答案，以及一份实验报告，内容围绕教学用操作系统 Unix V6++ 展开（进程/文件/用户管理、存储管理、虚拟存储器、进程间通信等）。

## 目录结构

```
ppts/      课程设计课件（围绕 Unix V6++ 的存储管理、虚拟存储器、
           进程通信基础及经典 IPC 问题等专题）
tasks/     作业及参考答案（进程与文件管理、存储管理理论、进程通信理论、
           死锁、历年期末考试题）
reports/   一份关于操作系统信号量（semaphore）的 LaTeX 报告
           （semaphore_os_report），包含编译好的 PDF 及 LaTeX 构建产物
```

### `ppts/`
课程设计系列课件（PDF/PPTX），内容包括：
- 离散化 Unix V6++ 的实现（请求调页式虚拟存储器）
- 物理内存管理
- 虚拟存储器
- 进程通信基础及经典 IPC 问题（含死锁）

### `tasks/`
作业及其参考答案，内容包括：
- Unix 进程、文件与用户
- 存储管理理论
- 进程通信理论（进程同步问题、死锁）
- 历年期末考试真题及参考答案（2024–2025、2025–2026）

### `reports/`
`semaphore_os_report` —— 一份关于操作系统信号量的 LaTeX 报告。
- `semaphore_os_report.pdf` —— 编译完成的报告（可正常打开阅读）
- `semaphore_os_report.tex` / `.aux` / `.toc` —— LaTeX 源文件及构建产物。
  **注意：** 这三个文件目前内容全部为空字节（磁盘上已损坏），只有编译好的
  PDF 是完整可用的。
- `algorithm2e.sty` —— 用于排版算法伪代码的 LaTeX 宏包

## 说明

- 大部分文件名及内容均为中文，与课程语言一致。
- 本仓库是个人课程资料归档，并非软件工程项目，课件与作业本身无需构建。
  `reports/` 中的 LaTeX 报告若源文件恢复后，可用 `pdflatex` 等工具重新编译。
